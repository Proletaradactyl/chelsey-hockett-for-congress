# Chelsey Hockett for Congress Campaign Site

A responsive political campaign website built with HTML, CSS, and JavaScript.

## Project Structure

```
chelsey/
├── index.html              Homepage
├── about.html              About page
├── issues.html             Issues & platform page
├── css/
│   └── style.css          Shared stylesheet
├── images/                Asset directory for photos and graphics
├── js/                     JavaScript functionality
└── README.md              This file
```

## Pages

- **index.html** – Homepage with hero, about preview, and key platforms
- **about.html** – Full biography, background, and candidate story
- **issues.html** – Detailed issue positions with expandable accordion

## Styling

All styles are organized in `css/style.css` using CSS custom properties (variables) for colors and typography. The site uses a teal and pink color palette with responsive design patterns.

## Colors

- Teal: `#3a8fa3` (primary)
- Pink: `#d4357a` (accent)
- Charcoal: `#1a2028` (text)

## Fonts

- **Oswald** – Headlines and labels
- **Satisfy** – Script/accent text
- **Source Sans 3** – Body text
